The **Ender Generator** burns certain End-themed items to produce FE.

## Configuration

The Ender Generator uses the `excessive_utilities:single_item_fuel` recipe type. Use this as an example:

```json
{
  "type": "excessive_utilities:single_item_fuel",
  "duration": 1600,
  "fe_per_tick": 40,
  "generator_type": "ender",
  "ingredient": {
    "tag": "c:ender_pearls"
  }
}
```
