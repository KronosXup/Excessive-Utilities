The **Halitosis Generator** burns Dragon Breath to produce FE.

## Configuration

The Halitosis Generator uses the `excessive_utilities:single_item_fuel` recipe type. Use this as an example:

```json
{
  "type": "excessive_utilities:single_item_fuel",
  "duration": 12000,
  "fe_per_tick": 40,
  "generator_type": "halitosis",
  "ingredient": {
    "item": "minecraft:dragon_breath"
  }
}
```
