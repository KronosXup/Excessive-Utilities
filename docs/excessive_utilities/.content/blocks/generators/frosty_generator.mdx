The **Frosty Generator** burns certain icy items to produce FE.

## Configuration

The Frosty Generator uses the `excessive_utilities:single_item_fuel` recipe type. Use this as an example:

```json
{
  "type": "excessive_utilities:single_item_fuel",
  "duration": 40,
  "fe_per_tick": 40,
  "generator_type": "frosty",
  "ingredient": {
    "item": "minecraft:ice"
  }
}
```
